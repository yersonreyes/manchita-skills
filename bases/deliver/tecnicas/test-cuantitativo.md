# Test Cuantitativo

## ¿Qué es?

El Test Cuantitativo es una técnica de validación que recopila datos numéricos y estadísticos sobre el desempeño del diseño. Mide métricas específicas como tasas de éxito, tiempo de tarea, satisfacción, etc.

## Cómo se usa

### Tipos de métricas:

| Métrica | Descripción | Ejemplo |
|---------|-------------|---------|
| Éxito | % de usuarios que completan la tarea | 85% completó checkout |
| Tiempo | Cuánto tarda en completar | 45 segundos promedio |
| Errores | Cuántos errores comete | 2 errores promedio |
| Satisfacción | Rating del usuario | NPS: +45 |

### Métodos:

- **Encuestas** — Post-task, SUS, NPS
- **Analytics** —行为 data del prototipo/producto
- **A/B Testing** — Comparar versiones
- **Encuestas masivas** — Validar alcance

### Proceso:

1. **Definir métricas** — Qué medir
2. **Diseñar instrumento** — Encuesta, task
3. **Reclutar participantes** — Muestra representativa
4. **Ejecutar** — Recopilar datos
5. **Analizar** — statistical analysis

### Diferencia con cualitativo:

| Cualitativo | Cuantitativo |
|-------------|--------------|
| Profundidad (por qué) | Amplitud (cuántos) |
| Insights rich | Datos hard |
| Pequeña muestra | Gran muestra |
