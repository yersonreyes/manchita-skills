# Storyboard

## ¿Qué es?

El Storyboard es una secuencia de dibujos o pantallas que cuenta una historia visual de cómo el usuario interactúa con el producto. Similar a los storyboards de cine, muestra el flujo de la experiencia.

## Cómo se usa

### Estructura:

- **Escenas secuenciales** — Paso a paso
- **Ilustraciones simples** — Qué pasa en cada paso
- **Anotaciones** — Qué siente el usuario
- **Notas técnicas** — Funcionalidades

### Usos comunes:

- Presentar flujos de usuario
- Planificar videos explicativos
- Validar journey completo
- Comunicar a stakeholders
