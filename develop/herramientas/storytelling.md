# Storytelling

## ¿Qué es?

El Storytelling es la técnica de comunicar ideas, conceptos o productos a través de historias. En diseño, se usa para presentar soluciones de manera memorable y emocional.

## Cómo se usa

### Elementos de una historia:

- **Personaje** — El usuario o protagonista
- **Problema** — El desafío a resolver
- **Viaje** — El proceso de diseño
- **Solución** — El producto o servicio
- **Beneficio** — El resultado positivo

### Usos:

- Presentar a stakeholders
- Comunicar el "por qué" del diseño
- Hacer memorable la solución
- Inspirar al equipo
