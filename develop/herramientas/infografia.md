# Infografía

## ¿Qué es?

La Infografía es una representación visual de información que combina texto, imágenes y gráficos para comunicar información compleja de manera clara y atractiva.

## Cómo se usa

### Usos:

- Presentar datos de investigación
- Explicar procesos
- Comunicar hallazgos
- Documentar el proyecto
