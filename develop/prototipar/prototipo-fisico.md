# Prototipo Físico

## ¿Qué es?

El Prototipo Físico es una representación tangible del producto en el mundo real, usando materiales físicos como cartón, foam, plástico, etc. Es especialmente útil para productos físicos, packaging, espacios o experiencias que requieren interacción con objetos reales.

## Cómo se usa

### Propósito:

- Validar forma, tamaño y ergonomía
- Testear interacción física
- Evaluar packaging y materiales
- Crear algo tangible para stakeholders

### Materiales comunes:

| Material | Uso | Costo |
|----------|-----|-------|
| Cartón | Estructura básica | Muy bajo |
| Foam | Textura y forma | Bajo |
| Madera balsa | Detalles | Bajo |
| Impresión 3D | Formas complejas | Medio |
| Resina | Acabado final | Alto |

### Niveles de fidelidad:

```
FÍSICO
├── Mockup de papel    → Cartón, básico
├── Modelo funcional   → Moving parts
├── Prototipo de usuario→ Near production
└── Producción         → Lo que se va a fabricar
```

### Proceso:

1. **Diseñar en 2D** — Planos, medidas
2. **Cortar/materiales** — Fabricar las partes
3. **Ensamblar** — Unir las piezas
4. **Testear** — Con usuarios reales
5. **Iterar** — based on feedback

### Ejemplos:

- Mockup de teléfono en cartón para testear tamaño
- Packaging de producto para validar en estantería
- Modelo de interfaz física para producto IoT

## Cuándo usarlo

- En **Develop** para productos físicos
- Para validar ergonomía
- En diseño de packaging
- Para espacios y retail
