# Prototipo para Empatizar

## ¿Qué es?

El Prototipo para Empatizar es una herramienta de diseño que crea representaciones tangibles o digitales del producto para entender mejor las necesidades, emociones y contexto del usuario. No busca validar funcionalidad, sino generar empatía.

## Cómo se usa

### Propósito:

- Entender el problema desde la perspectiva del usuario
- Sentir lo que el usuario siente
- Identificar моменты de fricción emocionales
- Validar assumptions antes de invertir en desarrollo

### Tipos:

| Tipo | Descripción | Fidelidad |
|------|-------------|-----------|
| Role-play | Actuar la experiencia | Baja |
| Bodystorming | Prototipar con el cuerpo | Baja |
| Environmental | Recrear el entorno | Media |
| Experiencial | Vivir la experiencia | Media-Alta |

### Ejemplos:

- **Role-play:** Simular ser un cliente en una tienda
- **Bodystorming:** Hacer cola para sentir la frustración
- **Environmental:** Replicar el escritorio del usuario

### Proceso:

1. **Definir qué empathy** — Qué queremos sentir
2. **Elegir el método** — Rol-play, environment, etc.
3. **Crear el prototipo** — Rápido y simple
4. **Interactuar** — Vivir la experiencia
5. **Documentar insights** — Lo que aprendimos

## Cuándo usarlo

- En **Develop** antes de diseñar soluciones
- Para validar que entendemos al usuario
- Cuando hay disconnect con el usuario
- En investigación de nuevos dominios
