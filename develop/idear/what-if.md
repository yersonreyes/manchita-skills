# What If

## ¿Qué es?

What If es una técnica de pensamiento divergente que usa preguntas hipotéticas "What if..." para explorar posibilidades extremas y desafiar assumptions. Estimula la creatividad permitiendo explorar escenarios sin límites.

## Cómo se usa

### Estructura de preguntas:

```
What if...
├── ...no existieran las limitaciones actuales?
├── ...el usuario pudiera hacer X?
├── ...pudiéramos eliminar Y?
├── ...la tecnología Z existiera?
└── ...cambiáramos el modelo de negocio?
```

### Tipos de What If:

| Tipo | Ejemplo | Propósito |
|------|---------|-----------|
| Inversion | ¿Y si elimináramos el login? | Simplificar |
| Extremo | ¿Y si fuera gratis? | Nuevos modelos |
| Tecnología | ¿Y si AI lo resolviera? | Innovación |
| Usuario | ¿Y si el usuario fuera X? | Nuevos segmentos |
| Competencia | ¿Y si competidora bajara 50%? | Estratégico |

### Proceso:

1. **Establecer el contexto** — Qué estamos diseñando
2. **Escribir preguntas What If** — 10-20 preguntas
3. **Explorar cada una** — Qué implicaría?
4. **Seleccionar las más interesantes** — Para prototipar

### Ejemplos:

| What If | Respuesta potencial |
|---------|---------------------|
| ¿Y si no hubiera passwords? | Login biométrico, magic links |
| ¿Y si la app trabajara offline? | Sync cuando hay conexión |
| ¿Y si pudiera pagar con cualquier cosa? | Crypto, tokens, swap |

## Cuándo usarlo

- En **Develop** para pensamiento lateral
- Para desafiar assumptions
- En sesiones de ideación
- Para encontrar oportunidades innovadoras
