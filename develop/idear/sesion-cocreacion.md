# Sesión de Cocreación

## ¿Qué es?

La Sesión de Cocreación es un taller donde usuarios, stakeholders y el equipo de diseño trabajan juntos para crear soluciones. El usuario no es solo el sujeto de investigación, sino co-creador del producto.

## Cómo se usa

### Preparación:

1. **Definir el objetivo** — Qué vamos a crear/resolver
2. **Seleccionar participantes** — Usuarios + equipo + stakeholders
3. **Preparar materiales** — Herramientas, espacio, estímulos
4. **Establecer reglas** — Respeto, colaboración, diversión

### Dinámica típica (2-4 horas):

```
00-30 min  → Warm-up y presentación
30-60 min  → Context setting (repasar findings)
60-90 min  → Generación de ideas (juntos)
90-120 min → Prototipado colaborativo
120+ min   → Presentación y feedback
```

### Roles en la sesión:

| Rol | Responsabilidad |
|-----|-----------------|
| Facilitador | Guiar el proceso, mantener energía |
| Recorder | Documentar ideas |
| Participantes | Aportar ideas, prototipar |
| Observador | Observar dinámicas (si es external) |

### Técnicas de cocreación:

- **Design charrettes** — Diseño intensivo en grupo
- **Collage** — Crear con imágenes
- **Card sorting** — Organizar información juntos
- **Prototyping hands-on** — Construir con las manos

### Después:

1. **Documentar todo** — Fotos, videos, notas
2. **Sintetizar** — Identificar mejores ideas
3. **Compartir** — Con el equipo más amplio
4. **Iterar** — 基于 feedback

## Cuándo usarlo

- En **Develop** para generar ideas con usuarios
- Para validar conceptos con usuarios reales
- Para build ownership con stakeholders
- Cuando el problema es complejo
