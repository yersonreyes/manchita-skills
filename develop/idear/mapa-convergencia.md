# Mapa de Convergencia

## ¿Qué es?

El Mapa de Convergencia es una herramienta visual que muestra cómo las ideas generadas en la fase de ideación se van reduciendo y focalizando hacia soluciones específicas. Es el proceso de ir de "muchas ideas" a "las mejores ideas".

## Cómo se usa

### Estructura:

```
DIVERGENCIA                              CONVERGENCIA
(muchas ideas)    ────────────────────>   (pocas ideas)

┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐
│ ● idea 1│   │         │   │         │   │         │
│ ● idea 2│   │ ●●●●●●●●│   │ ●●●●●   │   │ ●●●     │
│ ● idea 3│ ─→│ ideas   │ ─→│ cluster │ ─→│ top 3   │
│ ● idea 4│   │ 50+     │   │ por tema│   │ ideas   │
│   ...   │   │         │   │         │   │         │
└─────────┘   └─────────┘   └─────────┘   └─────────┘
```

### Proceso:

1. **Recopilar todas las ideas** — De brainstorming
2. **Agrupar por temas** — Affinity clustering
3. **Eliminar duplicados** — Consolidar similares
4. **Evaluar viabilidad** — Quick feasibility check
5. **Seleccionar top ideas** — Por criterios definidos

### Criterios de convergencia:

- **Viabilidad técnica** — Se puede construir?
- **Valor para usuario** — Resuelve el problema?
- **Valor de negocio** — Es rentable?
- **Diferenciación** — Es unique?

## Cuándo usarlo

- En **Develop** después de generar ideas
- Para priorizar y focalizar
- Antes de prototipar
- En workshops de ideación
