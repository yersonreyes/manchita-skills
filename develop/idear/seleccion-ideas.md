# Selección de Ideas

## ¿Qué es?

La Selección de Ideas es el proceso de evaluar y elegir las mejores ideas generadas para continuar desarrollando. Es el puente entre la generación divergente y la convergencia.

## Cómo se usa

### Framework de evaluación:

```
SELECCIÓN DE IDEAS
├── Criterios de Evaluación
│   ├── Impacto en usuario    (1-5)
│   ├── Viabilidad técnica    (1-5)
│   ├── Valor de negocio      (1-5)
│   ├── Facilidad de implementación (1-5)
│   └── Diferenciación       (1-5)
├── Matriz de priorización
└── Decisión final
```

### Métodos de selección:

1. **Votación simple** — Cada quien vote por sus favoritos
2. **Dot voting** — Dots limitados por persona
3. **Pareto** — 20% de ideas que generan 80% de valor
4. **Impact vs Effort** — Matriz 2x2
5. **Scorecard** — Puntuación ponderada

### Matriz Impacto-Esfuerzo:

```
                    ESFUERZO
                ┌──────────┬──────────┐
                │  BAJO    │  ALTO     │
        ┌───────┼──────────┼──────────┤
   ALTO │QUICK   │  🏆     │           │
        │WIN     │ PRIORITY│           │
        ├────────┼──────────┼──────────┤
   BAJO │  FILL  │  REJECT  │           │
        │  IN    │          │           │
        └────────┴──────────┴──────────┘
```

### Preguntas para evaluar:

- ¿Resuelve el problema principal?
- ¿Es técnicamente posible en timeline?
- ¿El usuario lo va a usar/pagar?
- ¿Tenemos recursos para implementarla?
- ¿Se diferencia de la competencia?

## Cuándo usarlo

- En **Develop** después de idear
- Para reducir scope a lo viable
- Antes de prototipar
- Para tomar decisiones de roadmap
