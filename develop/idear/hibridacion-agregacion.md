# Hibridación por Agregación

## ¿Qué es?

La Hibridación por Agregación es una técnica de ideación donde se combinan múltiples ideas o soluciones existentes en una nueva solución integrada. Es "sumar" elementos de diferentes fuentes.

## Cómo se usa

### Proceso:

1. **Identificar ideas base** — 2-3 ideas prometedoras
2. **Listar elementos de cada una** — Features, componentes
3. **Combinar** — Qué elementos funcionan juntos?
4. **Evaluar** — La híbrida tiene más valor?
5. **Iterar** — Refinar la combinación

### Ejemplo:

| Idea A | Idea B | Idea Híbrida |
|--------|--------|-------------|
| App de delivery | Suscripción mensual | Delivery + suscripción = pedidos recurrentes无忧 |
| Chat con soporte | FAQ inteligente | Chat humano + AI para escalamiento |

### Técnicas de agregación:

- **Feature stacking** — Sumar features
- **Best of each** — Tomar lo mejor de cada idea
- **Plus/Minus** — Agregar algo, quitar algo
- **Mashup** — Combinar dos productos/servicios

### Matriz de combinación:

```
        │ Idea B1 │ Idea B2 │ Idea B3 │
────────┼─────────┼─────────┼─────────│
Idea A1 │  A1+B1  │  A1+B2  │  A1+B3  │
Idea A2 │  A2+B1  │  A2+B2  │  A2+B3  │
Idea A3 │  A3+B1  │  A3+B2  │  A3+B3  │
```

## Cuándo usarlo

- En **Develop** para innovar
- Cuando hay ideas parciales
- Para crear propuesta de valor unique
- En workshops de ideación
