# Hibridación por Síntesis

## ¿Qué es?

La Hibridación por Síntesis es una técnica de ideación donde se combinan dos o más conceptos, teorías o elementos en un todo coherente y nuevo. A diferencia de la agregación (sumar features), aquí se busca crear algo fundamentalmente diferente.

## Cómo se usa

### Diferencia con otras hibridaciones:

| Tipo | Qué hace |
|------|----------|
| Agregación | Suma elementos existentes |
| Traslación | Trae solución de otro campo |
| **Síntesis** | Crea algo nuevo y coherente |

### Proceso:

1. **Identificar conceptos base** — Ideas fundamentales
2. **Encontrar puntos de conexión** — Dónde se tocan?
3. **Sintetizar** — Crear una nueva visión
4. **Validar** — Tiene sentido integrado?
5. **Desarrollar** — Detallar la híbrida

### Ejemplos:

| Concepto A | Concepto B | Síntesis |
|------------|------------|----------|
| Red social | Música | Spotify (descubrir música via social) |
| E-commerce | Suscripción | Amazon Prime (compra + beneficios) |
| Chat | AI | ChatGPT (conversación + inteligencia) |

### Niveles de síntesis:

1. **Superficial** — Combina features
2. **Estructural** — Cambia la arquitectura
3. **Paradigmática** — Nuevo modelo mental

## Cuándo usarlo

- En **Develop** para innovación disruptiva
- Cuando las soluciones actuales no funcionan
- Para crear categoría nueva
- En innovación de modelo de negocio
