# manchita-skills 🐾

> Repositorio de documentación de herramientas para el proceso de diseño UX/UI

## Acerca de

Este repo almacena documentación sobre herramientas utilizadas en las diferentes fases del proceso de diseño, siguiendo la metodología **Double Diamond**:

```
Discover → Define → Develop → Deliver
```

Las herramientas documentadas aquí no son herramientas digitales de diseño (Figma, Sketch, etc.), sino recursos, métodos y herramientas para el proceso de diseño en general.

---

## Estructura

```
manchita-skills/
├── discover/      # Herramientas para fase de descubrimiento
├── define/        # Herramientas para fase de definición  
├── develop/       # Herramientas para fase de desarrollo
├── deliver/       # Herramientas para fase de entrega
└── README.md
```

---

## Fases del Proceso de Diseño

### 🔍 Discover (Descubrir)
Investigación preliminar, empatizar con usuarios, entender el problema.

### 🎯 Define (Definir)
Síntesis de insights, definición del problema, creación de user personas.

### 💻 Develop (Desarrollar)
Ideación, prototipado, iteración de soluciones.

### 🚀 Deliver (Entregar)
Validación, lanzamiento, documentación final.

---

## Contribuir

1. Crear un archivo `.md` en la carpeta de la fase correspondiente
2. Incluir:
   - Nombre de la herramienta
   - Descripción breve
   - Cuándo usarla
   - Recursos/links útiles
