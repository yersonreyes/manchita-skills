# Matriz 2x2

## ¿Qué es?

La Matriz 2x2 es una herramienta de síntesis visual que clasifica elementos en dos dimensiones con dos valores cada una. Es útil para priorización, segmentación y toma de decisiones.

## Cómo se usa

### Estructura básica:

```
                    EJE Y (Dimensión 2)
                        │
              ALTO      │       ALTO
           ┌────────────┼────────────┐
           │            │            │
           │    ★★      │    ★★★     │
           │            │            │
    BAJO ──┼────────────┼────────────┼── BAJO
           │            │            │
           │     ★      │    ★★      │
           │            │            │
           └────────────┼────────────┘
                        │
                       EJE X (Dimensión 1)
```

### Matrices comunes:

| Matriz | Eje X | Eje Y |
|--------|-------|-------|
| Impacto-Esfuerzo | Esfuerzo | Impacto |
| Urgencia-Importancia | Urgencia | Importancia |
| Valor-Complejidad | Complejidad | Valor |
| Satisfacción-Necesidad | Necesidad | Satisfacción |

### Proceso:

1. **Definir dimensiones** — Qué vs Qué
2. **Listar items** — Features, problemas, etc.
3. **Evaluar cada uno** — En ambas dimensiones
4. **Posicionar** — En el cuadrante correcto
5. **Decidir** — Prioridad según cuadrante

## Cuándo usarlo

- En **Define** para priorización
- Para validar scope de proyecto
- En roadmap planning
- Para tomar decisiones de diseño
