# Insights Cluster

## ¿Qué es?

El Insights Cluster (Agrupación de Insights) es una técnica de síntesis donde se agrupan los insights de investigación en categorías o clusters temáticos para identificar patrones y prioritarlos.

## Cómo se usa

### Proceso:

1. **Recopilar todos los insights** — De entrevistas, encuestas, etc.
2. **Escribirlos en post-its** — Uno por insight
3. **Agrupar por temas** — Buscar patrones
4. **Nombrar los clusters** — Temas que emergen
5. **Priorizar** — Por impacto o frecuencia

### Estructura de un insight:

```
[INSIGHT]
─────────────────────────────────
Qué:        [Observación]
Por qué:    [Interpretación]
Impacto:    [Alto/Medio/Bajo]
Acción:     [Qué hacer con esto]
```

### Ejemplo de clusters:

```
┌──────────────────────────────────────────────────────────────┐
│                    INSIGHTS CLUSTERS                         │
├──────────────────────────────────────────────────────────────┤
│  CLUSTER: FACILIDAD DE USO                                   │
│  ┌────────────────────────────────────────────────────────┐  │
│  │ • Los usuarios no encuentran el botón de checkout      │  │
│  │ • La navegación es confusa para nuevos usuarios      │  │
│  │ • El contraste de colores afecta visibilidad         │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  CLUSTER: CONFIANZA                                          │
│  ┌────────────────────────────────────────────────────────┐  │
│  │ • Los usuarios dudan en meter datos de tarjeta        │  │
│  │ • Falta información de seguridad visible             │  │
│  └────────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────────┘
```

## Cuándo usarlo

- En **Define** para organizar findings
- Antes de priorizar features
- Para presentación a stakeholders
- En workshops de synthesis
