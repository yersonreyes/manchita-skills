# From-To

## ¿Qué es?

El diagrama From-To es una herramienta de síntesis que muestra la transformación o evolución de un estado actual (From) a un estado futuro (To). Ayuda a visualizar el cambio y las brechas a cerrar.

## Cómo se usa

### Estructura:

```
┌─────────────────────────────────────────────────────────────────┐
│                         FROM → TO                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│    FROM (Estado Actual)           TO (Estado Futuro)            │
│    ─────────────────────          ──────────────────           │
│                                                                  │
│    • Problema 1              →     • Solución 1                   │
│    • Problema 2              →     • Beneficio 2                  │
│    • Pain point X            →     • Oportunidad Y               │
│                                                                  │
│    [Contexto actual]                 [Visión de futuro]         │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Proceso:

1. **Definir el problema** — Estado From
2. **Definir la visión** — Estado To
3. **Mapear brechas** — Qué falta para llegar
4. **Identificar pasos** — Cómo hacer la transición
5. **Timeline** — En qué orden

### Ejemplo: App de banca

| From (Ahora) | To (Futuro) |
|--------------|-------------|
| Transferir toma 3 días | Transferencias instantáneas |
| Ir al banco físicamente | Todo desde el celular |
| Comisiones altas | Comisiones reducidas |

## Cuándo usarlo

- En **Define** para definir la visión del producto
- Para comunicación con stakeholders
- Para roadmap planning
- En workshops de estrategia
