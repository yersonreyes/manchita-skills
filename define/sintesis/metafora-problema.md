# Metáfora del Problema

## ¿Qué es?

La Metáfora del Problema es una técnica de síntesis que representa el problema de diseño usando una metáfora o analogía. Ayuda a entender el problema desde otra perspectiva y facilita la comunicación con stakeholders.

## Cómo se usa

### Proceso:

1. **Definir el problema** — Claro y específico
2. **Buscar metáforas** — Qué se parece a esto?
3. **Explorar la metáfora** — Qué insights da?
4. **Aplicar insights** — Volver al problema original

### Tipos de metáforas:

- **Del mundo físico** — Como un río con obstáculos
- **De la naturaleza** — Como una planta que necesita luz
- **De la vida cotidiana** — Como organizar un closet
- **De otros dominios** — Como un aeropuerto

### Ejemplos:

| Problema | Metáfora |
|----------|----------|
| Usuarios no completan onboarding | "Es como perder las llaves cada vez que salimos" |
| Sistema lento | "Es como esperar en una cola infinita" |
| Datos dispersos | "Es como tener un cuarto donde todo está tirado" |

### Beneficios:

- **Comunicación** — Más fácil de explicar a no-diseñadores
- **Perspectiva** — Ver el problema desde otro ángulo
- **Creatividad** — La metáfora sugiere soluciones
- **Empatía** — Conecta emocionalmente con el problema

## Cuándo usarlo

- En **Define** para crystalizar el problema
- En presentaciones a stakeholders
- Para generar ideas durante ideación
- Para hacer memorable el problema
