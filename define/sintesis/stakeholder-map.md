# Stakeholder Map (Síntesis)

## ¿Qué es?

El Stakeholder Map en síntesis es una herramienta visual para representar las relaciones, intereses y poder de influencia de todos los actores relacionados con un proyecto o producto. A diferencia del de investigación, aquí se enfoca en el análisis y estrategia.

## Cómo se usa

### Matriz de Poder/Interés:

```
                         INTERÉS
                    ┌───────────┬───────────┐
                    │  BAJO     │   ALTO    │
        ┌───────────┼───────────┼───────────┤
    A   │  KEEP     │  MANAGE   │           │
    L   │  SATISFIED│  CLOSELY  │           │
    T   │           │           │           │
    O   ├───────────┼───────────┼───────────┤
        │   BAJO    │           │           │
        │  INTEREST │  MONITOR  │   KEEP    │
        │           │           │  INFORMED │
        └───────────┴───────────┴───────────┘
```

### Categorías de stakeholders:

| Categoría | Acción |
|-----------|--------|
| **Manage Closely** | Involucrar en decisiones, mantener informados |
| **Keep Satisfied** | Actualizar regularmente, evitar sorpresas |
| **Keep Informed** | Consultar, compartir información |
| **Monitor** | Observar, mínimo esfuerzo de comunicación |

### Tipos de relaciones a mapear:

- **De influencia** — Quién puede afectar el proyecto
- **De dependencia** — Quién necesita qué
- **De comunicación** — Cómo se comunica con cada uno
- **De decisión** — Quién tiene poder de veto/aprobación

## Cuándo usarlo

- En **Define** para planificar comunicación
- Antes de presentar a clientes/stakeholders
- Para identificar bloqueos potenciales
- En gestión de proyectos
