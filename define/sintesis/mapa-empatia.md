# Mapa de Empatía (Síntesis)

## ¿Qué es?

El Mapa de Empatía es una herramienta visual que captura las perspectivas y emociones de un usuario específico. En Síntesis, se crea/refina después de investigación para representar al usuario objetivo.

## Cómo se usa

### Estructura:

```
┌──────────────────────────────────────────────────────────────────┐
│                          USUARIO: [Nombre]                      │
├────────────────────┬────────────────────────────────────────────┤
│     VE             │            OYE                              │
│  Qué ve en         │  Qué escucha el                            │
│  su entorno?       │  usuario?                                 │
│                    │                                            │
├────────────────────┼────────────────────────────────────────────┤
│     PIENSA         │            SIENTE                          │
│  Qué piensa el    │  Qué emociones                            │
│  usuario?         │  experimenta?                             │
│  (miedos,         │  (frustración,                             │
│   creencias)      │   motivación, alegría)                      │
├────────────────────┼────────────────────────────────────────────┤
│     DICE           │            HACE                             │
│  Qué dice el      │  Qué hace el                              │
│  usuario?         │  usuario?                                 │
│                    │                                            │
└────────────────────┴────────────────────────────────────────────┘
```

### Proceso:

1. **Elegir una persona** — De las creadas
2. **Completar cada cuadrante** — Con findings de research
3. **Validar** — Asegurar que sea real y verificable
4. **Compartir** — Con el equipo

## Cuándo usarlo

- En **Define** para profundizar en usuario
- Para validar assumptions
- En workshops de empathy
- Antes de ideación
