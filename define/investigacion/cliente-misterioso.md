# Cliente Misterioso (Mystery Shopper)

## ¿Qué es?

El Cliente Misterioso (Mystery Shopping) es una técnica de investigación donde una persona entrenada evalúa un servicio, producto o proceso actuando como un cliente común. Permite vivir la experiencia desde la perspectiva del usuario.

## Cómo se usa

### Proceso:

1. **Definir criterios** — Qué evaluar (UX, servicio, proceso)
2. **Crear guía de evaluación** — Checklist de observación
3. **Seleccionar escenarios** — Qué tareas hacer
4. **Ejecutar** — Visit/app/llamada como cliente normal
5. **Documentar** — Notas detalladas post-visita
6. **Analizar** — Identificar issues y oportunidades

### Áreas a evaluar:

- **Tiempo de respuesta** — Cuánto espera
- **Facilidad de uso** — Intuitividad
- **Calidad de atención** — Amabilidad, resolución
- **Consistencia** — Misma experiencia en diferentes canales
- **Pain points** — Dónde se frustra el usuario

### Después del Mystery Shop:

```
REPORTE:
- Fecha y hora
- Canal (web, app, tienda, teléfono)
- Escenario ejecutado
- Pasos realizados
- Tiempos
- Issues encontrados
- Fotos evidencia
- Score general
```

## Cuándo usarlo

- En **Define** para entender experiencia actual
- Para audit de servicio al cliente
- En evaluación de competidores
- Para encontrar problemas ocultos
