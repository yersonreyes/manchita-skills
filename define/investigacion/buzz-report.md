# Buzz Report

## ¿Qué es?

El Buzz Report es un informe que documenta y analiza el "ruido" (buzz) en medios y redes sociales sobre un tema, marca, producto o competencia. Mide el sentiment, volumen de menciones y tendencias.

## Cómo se usa

### Componentes del Buzz Report:

1. **Volumen de menciones** — Cuánto se habla
2. **Sentiment** — Positivo/Negativo/Neutral
3. **Reach** — Cuántas personas alcanzaron
4. **Share of Voice** — vs competidores
5. **Key themes** — Temas más mencionados
6. **Influencers** — Quiénes lideran la conversación

### Herramientas para recopilar:

| Herramienta | Funcionalidad |
|-------------|---------------|
| Google Alerts | Menciones gratuitas |
| Mention | Mencias en tiempo real |
| Brandwatch | Social listening avanzado |
| BuzzSumo | Contenido trending |

### Estructura del reporte:

```
# Buzz Report: [Tema]
## Resumen Ejecutivo
## Volumen Total
## Sentiment Analysis
## Temas Principales
## Competencia
## Conclusiones
## Recomendaciones
```

## Cuándo usarlo

- En **Define** para validar interés de mercado
- Para entender percepción de marca
- Antes de lanzar producto
- En monitoring de campaña
