# Focus Group

## ¿Qué es?

El Focus Group (Grupo Focal) es una técnica de investigación cualitativa donde un grupo de 6-10 participantes discute un tema específico moderado por un facilitador. Permite observar interacciones, debates y reacciones grupales.

## Cómo se usa

### Preparación:

1. **Definir objetivos** — Qué queremos aprender
2. **Seleccionar participantes** — Perfil específico (mismo segmento)
3. **Crear guía de discusión** — Temas y preguntas
4. **Preparar materiales** — Prototipos, productos, stimuli
5. **Elegir lokasi** — Físico o virtual

### Dinámica típica (60-90 min):

```
00-10 min   → Introducción y icebreaker
10-20 min   → Warm-up: preguntas generales
20-45 min   → Discusión principal: stimuli, preguntas
45-60 min   → Síntesis y conclusion
```

### Rol del moderador:

- Facilitar la discusión
- Asegurar que todos participen
- Mantener el foco en los objetivos
- Observar dinámicas grupales
- No influir en las respuestas

### Después del focus group:

1. **Tomar notas exhaustivas** — Observaciones, no solo quotes
2. **Revisar grabación** — Si aplica
3. **Identificar patrones** — Qué se repetía
4. **Separar lo individual vs grupal** — Comments vs reactions

## Cuándo usarlo

- En **Define** para validar conceptos
- Para entender percepciones de marca
- Para probar prototipos o mensajes
- Cuando se necesita debate y diversidad de perspectivas
