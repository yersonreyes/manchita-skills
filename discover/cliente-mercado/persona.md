# User Persona

## ¿Qué es?

Una User Persona es una representación arquetípica de un usuario objetivo, basada en datos reales de investigación. Es un personaje ficticio pero fundamentado que ayuda a tomar decisiones de diseño centrado en el usuario.

## Cómo se usa

### Componentes de una Persona:

```
┌─────────────────────────────────────────────┐
│              [Nombre de la Persona]        │
│            (Una foto representativa)        │
├─────────────────────────────────────────────┤
│  DEMOGRAFICOS                               │
│  - Edad:                                    │
│  - Profesión:                               │
│  - Ubicación:                               │
│  - Ingresos:                                │
├─────────────────────────────────────────────┤
│  BIO / HISTORIA                             │
│  (Contexto personal y profesional)         │
├─────────────────────────────────────────────┤
│  METAS / MOTIVACIONES                       │
│  - Qué quiere lograr                        │
│  - Por qué le importa                       │
├─────────────────────────────────────────────┤
│  FRUSTRACIONES / PAIN POINTS               │
│  - Qué le frustra                           │
│  - Barreras que enfrenta                    │
├─────────────────────────────────────────────┤
│  COMPORTAMIENTO                             │
│  - Cómo usa tecnología                     │
│  - Canales preferidos                      │
├─────────────────────────────────────────────┤
│  QUOTE                                      │
│  (Cita representativa del usuario real)    │
└─────────────────────────────────────────────┘
```

### Tipos de Personas:

1. **Primary** — Usuario principal, diseñar para esta
2. **Secondary** — Usuario secundario, considerar
3. **Adverse** — A quien NO sirve el producto
4. **Stakeholder** — Usuarios de negocio

## Cuándo usarlo

- En **Discover** después de investigar usuarios
- En **Define** para crystalizar el target
- En **Develop** para guiar decisiones de diseño
- Para comunicar insights al equipo
