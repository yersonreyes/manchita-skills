# Stakeholder Map

## ¿Qué es?

El Stakeholder Map es una herramienta visual que identifica y clasifica a todas las personas, grupos o entidades que tienen interés o influencia en un proyecto, producto o iniciativa.

## Cómo se usa

### Matriz de Stakeholders:

```
                        INFLUENCIA
                    ┌───────────┬───────────┐
                    │  MEDIA    │   ALTA    │
        ┌───────────┼───────────┼───────────┤
    ALTA│  KEEP     │  MANAGE   │           │
        │  SATISFIED│  CLOSELY  │           │
        ├───────────┼───────────┼───────────┤
    P   │           │  MONITOR  │   LOW     │
    O   │   LOW     │           │  KEEP     │
    W   │  INTEREST│           │  INFORMED │
    E   ├───────────┼───────────┼───────────┤
    R   │  LOW      │   MEDIA   │           │
        │           │           │           │
        └───────────┴───────────┴───────────┘
```

### Clasificación:

| Categoría | Qué hacer |
|-----------|------------|
| **Manage Closely** | Involucrar activamente, comunicar seguido |
| **Keep Satisfied** | Satisfacer necesidades, evitar conflictos |
| **Keep Informed** | Informar regularmente, escuchar feedback |
| **Monitor** | Observar, mínimo esfuerzo |

### Paso a paso:

1. **Listar** todos los stakeholders potenciales
2. **Clasificar** por influencia y poder
3. **Identificar relaciones** entre ellos
4. **Definir estrategia** por cada grupo

## Cuándo usarlo

- En **Discover** para mapear el ecosistema
- Al inicio de cualquier proyecto
- Para planificación de comunicación
- En proyectos con múltiples dependencias
