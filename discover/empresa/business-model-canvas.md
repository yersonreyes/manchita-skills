# Business Model Canvas

## ¿Qué es?

El Business Model Canvas (BMC) es una herramienta estratégica que permite visualizar, diseñar y reinventar modelos de negocio en una sola página. Fue creada por Alexander Osterwalder.

## Cómo se usa

Se compone de **9 bloques** que se llenan en este orden sugerido:

```
┌─────────────────┬─────────────────┬─────────────────┐
│   VALUE PROPS    │    CUSTOMERS    │    REVENUE      │
│   (Propuesta    │    (Clientes)   │    (Ingresos)   │
│    de valor)    │                 │                 │
├─────────────────┴─────────────────┼─────────────────┤
│           RESOURCES               │    CHANNELS     │
│         (Recursos)                │    (Canales)    │
├────────────────────────────────────┼─────────────────┤
│              ACTIVITIES            │  RELATIONSHIPS  │
│            (Actividades)           │   (Relaciones)  │
├────────────────────────────────────┴─────────────────┤
│                 COST STRUCTURE                      │
│              (Estructura de costos)                 │
└────────────────────────────────────────────────────┘
```

### Los 9 bloques:

1. **Value Propositions** — Propuesta de valor
2. **Customer Segments** — Segmentos de clientes
3. **Customer Relationships** — Relaciones con clientes
4. **Channels** — Canales de distribución
5. **Revenue Streams** — Flujos de ingresos
6. **Key Resources** — Recursos clave
7. **Key Activities** — Actividades clave
8. **Key Partnerships** — Alianzas clave
9. **Cost Structure** — Estructura de costos

## Cuándo usarlo

- Al inicio de un proyecto para entender el contexto de negocio
- En **Discover** para mapear el ecosistema actual
- En **Define** para refinar la propuesta de valor
- Para validar ideas de producto/servicio
- En sesiones de strategy con stakeholders
