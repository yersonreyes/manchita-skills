# Customer Journey Map

## ¿Qué es?

El Customer Journey Map (Mapa de Viaje del Cliente) es una representación visual de la experiencia de un usuario a lo largo del tiempo, desde que descubre un producto/servicio hasta que lo usa y más allá. Muestra las etapas, acciones, emociones y touchpoints.

### Componentes clave:

1. **Etapas** — Fases del viaje (Awareness, Consideration, Purchase, Use, Loyalty)
2. **Acciones** — Qué hace el usuario en cada etapa
3. **Emociones** — Cómo se siente (escalera emocional)
4. **Touchpoints** — Puntos de contacto con la marca
5. **Pain points** — Frustraciones y problemas
6. **Opportunities** — Oportunidades de mejora

## Cómo crearlo:

1. **Definir el escenario** — Qué journey vas a mapear
2. **Elegir el usuario** — Usar una Persona
3. **Investigar** — Datos de usuarios reales
4. **Mapear** — Visualizar cada etapa
5. **Validar** — Con usuarios reales

## Cuándo usarlo

- En **Discover** para entender la experiencia actual
- En **Define** para identificar problemas a resolver
- En **Develop** para diseñar soluciones
- Para identificar gaps entre canales
